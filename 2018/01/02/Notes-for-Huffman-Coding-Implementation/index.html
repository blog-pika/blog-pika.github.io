A Note for Scala Sequence Pattern Matching

This is a mistake I made when doing the coding assignment from course Functional Programming Principles in Scala. The assignment required us to implement Huffman coding in a functional way.

For descriptions please refer to the question I asked in Stack Overflow: Scala Pattern Matching Error, “bad simple pattern: bad use of _* (sequence pattern not allowed)”.

The lesson is that:

If you want to match against a sequence without specifying how long it can be, you can specify _* as the last element of the pattern.

Unfortunately, I used _* at the head of the match.
